## 📝 Introduction / 简介

**Abstract:**
Detecting AI-generated images (AIGC) is becoming increasingly challenging. Existing methods often struggle with generalization and lack interpretability. In this work, we propose **AgentFoX**, a novel framework that leverages Large Language Model (LLM) agents to guide the fusion of multi-modal features. By incorporating an explainable reasoning mechanism, AgentFoX not only achieves state-of-the-art detection performance but also provides textual explanations for its decisions.

**摘要：**
检测 AI 生成图像（AIGC）正变得日益在挑战性。现有的方法往往在泛化性上表现挣扎，且缺乏可解释性。在这项工作中，我们提出了 **AgentFoX**，这是一个利用大语言模型（LLM）智能体来指导多模态特征融合的新颖框架。通过引入可解释的推理机制，AgentFoX 不仅实现了最先进的（SOTA）检测性能，还能为其决策提供文本解释。
